## git
### 上传文件
 ```
  git init #初始化
  touch README.md #增加readme文件
  git add . 
  git commit -m "add code" #增加描述文件
  git remote add origin https://github.com/luyajun01/code.git #将修改好的文件上传至github 
  git push origin master #上传文件

```
